what is HTML

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser

History

HTML history in brief:

  1. 1989–1991 – Origins
    • Created by Tim Berners-Lee at CERN.
    • Designed for sharing scientific documents over the web.
    • First version included basic tags like headings, paragraphs, and links.


  2. 1995 – HTML 2.0
    • First standardized version by the IETF.
    • Added forms and improved interoperability.


  3. 1997 – HTML 3.2
    • Introduced tables, applets, and richer page formatting.
    • Browser competition (“browser wars”) heavily influenced development.


  4. 1997–1999 – HTML 4.0 / 4.01
    • IAdded CSS support, scripting, frames, and accessibility improvements.
    • Encouraged separating content from presentation.


  5. 2000 – XHTML
    • HTML rewritten using XML rules.
    • EStricter syntax requirements.
    • Adoption was mixed because it was harder for developers.


  6. 2008–2014 – HTML5
    • Developed by the WHATWG and W3C.
    • Added multimedia support (
    • , canvas graphics, local storage, and APIs for modern web apps.
    • Became the official standard in 2014.


  7. Modern HTML (HTML Living Standard)
    • Maintained continuously by WHATWG.
    • No more major numbered releases.
    • BUpdated incrementally to support evolving web technologies.
